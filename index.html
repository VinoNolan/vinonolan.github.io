About Me

Saya adalah mahasiswa di salah satu universitas swasta surakarta prodi Teknik Informatika. Walaupun sudah masuk di semester 6 saya mengakui bahwa masih banyak sekali kekurangan di dalam diri saya terutama dalam kompetensi saya sebagai seorang mahasiswa TI. Kendati demikian saya semangat berjuang dan terus belajar dan mengasah skill kompetensi saya khususnya dalam bidang Front end web development.

Saya memiliki pengalaman PKL di toko reparasi laptop/komputer di daerah mendungan sukoharjo selama 4.5 bulan, dari sana saya memiliki skill dalam troubleshooting software khususnya windows. Saya juga memiliki pengalaman pekerjaan entry data di BPN Boyolali. Sekitar 1 bulan sudah saya habiskan untuk latihan dan belajar setiap hari dalam materi front end di channel youtube web programming unpas.
